# FullHDFlix_OfficialBot Setup Guide

1. Install Python (3.10+ recommended)
2. Install dependencies:
   pip install -r requirements.txt

3. Set your config in config.py:
   - BOT_TOKEN
   - GROUP_ID
   - CHANNEL_ID
   - FORCE_JOIN_YT
   - FORCE_JOIN_IG

4. Run the bot:
   python main.py

Hosting Options:
- Replit (upload all files, run main.py)
- Render.com (create web service, add start command: python main.py)
